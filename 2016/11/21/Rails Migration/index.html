create_table方法

可选参数:

  1. id: 是否需要一个自增的id
  2. primary_key: 声明主键字段
  3. force: 在创建之前先把表删除，默认为false，也可设置为cascade，会删除关联表
  4. options: 其他的选项

column方法

可选参数

  1. limit: 长度
  2. default: 默认值
  3. null: 是否允许为空
  4. index: 建立该字段的索引
  5. precision, scale: 声明float时候可用

Integer 参数

index方法

新建一个索引
t.index [:index], name: "index_name"

references方法
